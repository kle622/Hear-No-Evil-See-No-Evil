----- Debug Modes -----

Press 'space' to toggle the free-floating debug camera. While in debug camera
mode, press 'o' to decrease camera movement speed, and press 'p' to increase it.
The 'j' and 'k' keys can be used to move the camera vertically down and up,
respectively.

Press 'b' to toggle debug geometry drawing in normal camera mode. Geometry
drawing will always be present in debug camera mode.
